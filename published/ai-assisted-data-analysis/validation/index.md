# Validation

How this bundle is validated — sourced claims are checked, and each validation sweep is recorded here as a dated audit.

This bundle is `citation_style: deeply-cited` (see `research-profile.yaml`): claims about model capabilities and limits — context-window sizes, whether a given tool runs real code against a file, documented rates of numerical or arithmetic error — are held to the same standard as its primary sources, not summarized secondhand. No claim ships until it traces to a fetched primary source in `evidence.yaml`. Audits of specific pages, once authored, are recorded in `audits/` here.
