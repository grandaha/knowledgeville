# Validation

This bundle's validation posture is **selective citation with verify-embedded-facts**.
That means we cite figures that appear in the primary source and verify the ones we
repeat. We don't independently replicate the analyst's research — we check that numbers
quoted in a brief match what the source actually says.

## The defining rule

An analyst piece is a primary source for its own figures. When we quote a number from
the piece, we cite the piece. When the piece names a third-party source for a figure,
we trace it there and cite the original. When we can't trace it, we say so plainly.

Noting an author's commercial stake is not the same as calling a figure unsourced.
Stake is a signal to read carefully — it goes in the "How much to trust it" section,
not in the citation record.

## Temporal posture

These briefs are **point-in-time-supersedable**: each captures one analyst's argument as
it stood when we wrote the brief. A later piece doesn't revise an old brief — it gets
its own. Briefs are dated at the top so readers know exactly when the snapshot was taken.

## Audits

Dated validation sweeps are recorded in the `audits/` directory. Each sweep is run by
the citation-auditor agent and covers the figures, attributions, and sourcing decisions
for the brief in scope.
