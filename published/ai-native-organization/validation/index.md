# Validation

How this bundle is validated — sourced claims are checked, and each validation sweep is
recorded here as a dated audit.

This bundle argues a thesis about where organizations are heading, so its citation bar is
higher than a purely descriptive bundle: every load-bearing statistic must trace to a
primary source (never a model's recalled citation, never an unverified web-search
summary), and every claim's status in `evidence.yaml` is checked against that source
directly before it ships. Because this is a live, fast-moving topic, claims are
point-in-time-supersedable — a newer study that changes the picture supersedes an older
one rather than being added alongside it indefinitely.
