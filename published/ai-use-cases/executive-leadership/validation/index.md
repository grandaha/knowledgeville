# Validation

This bundle is **demo-first**: most pages are runnable prompts and guided workflows, not cited essays, so the proof is in the before-and-after rather than a footnote. But any fact a page embeds about a tool or platform — a model's capability, a feature, a limit — is verified against that platform's own documentation before it ships.

Each validation sweep is recorded here as a dated audit. The bundle's posture is defined in [research-profile.yaml](/ai-use-cases/executive-leadership/research-profile.yaml), and verified claims live in [evidence.yaml](/ai-use-cases/executive-leadership/validation/evidence.yaml).
