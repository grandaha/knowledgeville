# Validation

These pages are **demo-first**: the value is a prompt you run, not a set of cited statistics,
so most pages carry no citations. That does not mean nothing is checked.

Every **embedded fact** is verified against a primary source before a page ships. That covers
two kinds of fact in this bundle:

- **Platform facts** — a post length or hashtag cap in a marketing recipe is checked against
  the platform's own help docs (the Instagram lesson: a recipe that states a limit must state
  the right one).
- **Rules an agent can't get wrong** — where a recipe names a fair-housing constraint, an MLS
  rule, or a do-not-call boundary, that guidance is checked against a current primary source,
  not written from memory.

Each validation sweep is recorded here as a dated audit under `audits/`.
